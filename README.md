# СБР Калькулятор резиновых покрытий

Веб-приложение для расчёта расхода материалов и стоимости резиновых покрытий из СБР-крошки: чёрный и окрашенный слои, пигмент, клей, праймер, скипидар и работы.

## Возможности

- Расчёт площади вручную или по длине и ширине участка
- Один слой (чёрная крошка) или два (чёрный + окрашенный с пигментом)
- Пропорциональный расход клея по общей толщине покрытия
- Запас материала в процентах
- Итоговая смета с ценой за м²
- Экспорт результата в Excel
- Светлая и тёмная тема (сохраняется в браузере)

## Быстрый старт

Сервер не обязателен — достаточно открыть в браузере файл `index.html` в корне проекта.

```text
index.html              — приветственная страница (лендинг)
calculator/index.html   — калькулятор
assets/css/             — стили
assets/js/              — скрипты темы и расчёта
```

### Локальный сервер (по желанию)

Если нужны корректные относительные пути при разработке:

```bash
# Python 3
python -m http.server 8080

# или Node.js (npx)
npx serve .
```

Откройте в браузере: `http://localhost:8080`
